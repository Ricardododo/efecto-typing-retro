# Efecto Typing Retro

Efecto de máquina de escribir estilo retro CRT con HTML, CSS y JavaScript.

## Características

- Efecto de escritura tipo máquina de escribir
- Estilo visual CRT/retro con efectos de scanline
- Cursor parpadeante
- Diseño responsive
- Palabras personalizadas configurables

## Uso

Abre `typing_retro.html` en tu navegador o visita la versión online en GitHub Pages.

## Configuración

Edita el array `phrases` en el script para cambiar los textos:

```javascript
const phrases = [
    "Tu texto 1",
    "Tu texto 2",
    "Tu texto 3"
];
```
